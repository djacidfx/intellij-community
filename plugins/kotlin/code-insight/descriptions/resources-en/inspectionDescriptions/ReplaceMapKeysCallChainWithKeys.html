Reports map { it.key }.toSet() call chains on map receivers that can be replaced with keys.

The quick-fix replaces the call chain with the map's keys property.

Example:


  fun test(steps: Map<String, Int>) {
      steps.map { it.key }.toSet()
  }

After the quick-fix is applied:


  fun test(steps: Map<String, Int>) {
      steps.keys
  }
